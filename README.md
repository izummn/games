# games
fifteen_puzzle

Manual.

The main goal is collect a sequence of numbers from 1 to 15. To move a number into an empty cell, use the keys:
"w" - move up;
"s" - move down;
"a" - move left;
"d" - move right.

To create a new game  use the key "n".
To restart the game use the key  "r".
To exit from the game - "q".
